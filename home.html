HTML QUESTIONS

Q1:What is HTML?

HTML stands for HyperText Markup Language. It is the standard language used to create and design webpages.

Q2: What is the use of the tag?

The tag defines a hyperlink, which is used to link from one page to another. It uses the href attribute to define the link destination. Go to Google

Q3: What is the use of the tag?

The tag is used to display images in an HTML page. It requires the src attribute (source of the image) and alt (alternative text if the image doesn't load).

Q4: What is the difference between id and class in HTML?

        | Feature        | `id`                       | `class`                    |
        | -------------- | -------------------------- | -------------------------- |
        | **Uniqueness** | Must be unique in a page   | Can be used multiple times |
        | **Use case**   | For single element styling | For group element styling  |
        | **Selector**   | `#idname` in CSS           | `.classname` in CSS        |

Q5: What is Responsive Web Design?

Responsive Web Design (RWD) is an approach that makes web pages look good on all devices (desktops, tablets, phones). It uses flexible layouts, images, and CSS media queries to adapt the page to different screen sizes.
